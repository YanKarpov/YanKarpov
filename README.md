## 👨‍💻 Обо мне

Я считаю, что программирование - это не просто профессия, а способ создавать решения, которые делают жизнь удобнее и интереснее.

В работе и учебе я придерживаюсь принципа **Per aspera ad astra** - через трудности к звёздам. Это помогает мне не сдаваться и постоянно расти как разработчику.

Сейчас я развиваюсь в направлении **fullstack-разработки**:
- Backend: Python (боты, REST API, автоматизация), JavaScript (Node.js, работа с API)
- Frontend на React
- Работа с базами данных и архитектурой приложений

Мне важно не просто писать код, а создавать **полезные и понятные решения**, которые реально могут использовать люди.
